Trees from 353 data

This tree was generated from reads from genome skim data subsetted based on their alignment to 353 loci. The reads were then assembled in HybPiper. The tree was constructed using IQTree. Node support is given as Bootstrap/gCF/sCF.

Trees from SISRS data

This tree was generated from reads from genome skim data. Variable sites were obtained using SISRS. The tree was constructed using RAxML with a dataset limited to biallelic sites, sites without gaps, and limited to a maximum of five samples missing per site. Node support is given as Bootstrap.
